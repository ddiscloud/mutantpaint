# 🎨 Mutant Paint

외형과 스킬을 진화시키는 크리처 육성 게임

## 🎮 빠른 시작 (로컬 실행)

```bash
# 패키지 설치
pip install -r requirements.txt

# 게임 실행
streamlit run streamlit_app.py
```

브라우저에서 http://localhost:8501 접속

---

## 📘 게임 가이드

### 🧬 개체 시스템
- 능력치: HP, ATK, MS (전투력에 영향)
- 외형: 메인/서브/패턴 색상 + 패턴 구성
- 스킬: 3개 슬롯(회복/공격/보조)으로 전투 스타일 구성
- 등급: Normal → Rare → Epic → Unique → Legendary → Mystic

### 🔄 믹스 (교배)
- 2개체를 선택해 유전 조합으로 새로운 개체 생성
- 돌연변이 확률로 상위 등급 외형/스킬을 획득 가능
- 부모의 능력치를 일부 계승하며 전투력이 더 높아질 수 있음

### 📦 랜덤 박스
- 하루 1회 개체를 무료로 획득
- 일반 교배보다 희귀 등급이 나올 확률이 높음

### ⚔️ 전투 시스템
- 1:1 턴제 자동 전투 (행동 게이지: MS 기반)
- 3개 스킬을 활용해 공격/회복/보조 전략을 구성
- 스테이지 클리어 시 보상 획득, 점진적 난이도 상승

### 📚 도감
- 수집한 색상/패턴/스킬을 기록하여 컬렉션 진행도를 확인
- 희귀 항목을 목표로 수집을 확장

### ⭐ 플레이 팁
